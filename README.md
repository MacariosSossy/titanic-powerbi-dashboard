# Titanic Survival Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing survival patterns from the Titanic dataset.

## Key Insights
- First-class passengers had the highest survival rates
- Female passengers were significantly more likely to survive
- Higher ticket fares correlate with increased survival probability

## Tools Used
- Power BI
- Data Visualization
- Exploratory Data Analysis
